# **FETCH: ANDROID UI SPECIFICATION**

Module C: Client-Side Interaction & UI  
Classification: INTERNAL / PRIVATE  
Version: 4.0.0 (The "Terrier" Revision)

# **1\. VISUAL SYSTEM: "TERRIER MONOCHROME"**

**Axiom of Calm:** The interface must not compete for attention. It is a utility, not a casino.

## **1.1. Color Palette**

* **Canvas:** Pure White (\#FFFFFF).  
* **Surfaces:** Off-White / Bone (\#F9F9F7). Used for cards, bubbles, and panels.  
* **Ink:** Jet Black (\#1A1A1A). Used for primary text.  
* **Graphite:** Dark Grey (\#4A4A4A). Used for secondary text/icons.  
* **Alerts:**  
  * *Urgent:* Crimson (\#D32F2F).  
  * *Warning:* Amber (\#FFA000).  
  * *Safe:* Emerald (\#388E3C).  
* **Constraint:** **NO** Teal or Orange. The interface is strictly monochrome except for semantic alerts.

## **1.2. Typography**

* **Primary (UI):** System Sans-Serif (Clean, legible).  
* **Persona (Fetch):** *Crimson Text* (Google Fonts) or similar high-readability Serif. Used *exclusively* for dialogue bubbles generated by Archimedes.

# **2\. MAIN SCREEN: "THE DOG HOUSE"**

Route: app/(home)/index.tsx  
Layout: Safe Area View with absolute positioning for the background.

## **2.1. The Background**

* **Asset:** watermark\_terrier\_outline.svg  
* **Position:** Center-aligned, 80% width.  
* **Opacity:** 0.03 (Barely perceptible).  
* **Behavior:** Static. Does not scroll with the chat content.

## **2.2. The Header (components/DogHouseHeader.tsx)**

A custom navigation bar, pinned to the top.

* **Left: The Hamburger**  
  * **Icon:** Simple 3-bar menu.  
  * **Action:** Opens the **Settings Drawer** (Side Modal).  
  * **Drawer Contents:**  
    * *Sass Level Indicator* (Read-only, 15%).  
    * *Theme Toggle* (Light/Dark).  
    * *House Tools:* "Coop Cam" (Shows "Under Development" Popup), "Server Health".  
* **Center: The Model Toggle**  
  * **Component:** Small Dropdown / Picker.  
  * **Options:**  
    * **Fetch (Public):** Routes to Google Gemini via Wrapper. (Icon: Cloud).  
    * **Fetch (Private):** Routes to Local DeepSeek-R1. (Icon: Shield).  
  * **Visual:** Minimal text, chevron down.  
* **Right: The Notification Badge**  
  * **Icon:** icon\_card\_stack\_bubble.svg (Miniaturized speech bubble containing stacked cards).  
  * **Logic:**  
    * *No Drafts:* Icon is outlined graphite.  
    * *Drafts Exist:* Icon is filled.  
    * *Badge:* Small circle overlay with count.  
    * *Color:*  
      * **Red:** High Priority (Financial/Medical/Urgent).  
      * **Yellow:** Standard correspondence.  
      * **Green:** Low priority / Newsletters.

## **2.3. The Body (components/ChatStream.tsx)**

* **Component:** FlashList.  
* **Content:** A linear history of the conversation with Fetch.  
* **Avatar Placement:** The 3D/2D Avatar is **Sticky** at the top of the list (or integrated into the Header area depending on scroll position).  
* **Interaction:**  
  * **"The Poke":** Tapping the Avatar (or the Notification Badge) triggers the **Deck Modal**.

# **3\. MODAL INTERFACE: "THE DECK"**

Context: The decision engine. Triggered by poking the dog.  
Presentation: Full-screen modal or heavy bottom-sheet (90% height).

## **3.1. State A: Active Notifications**

* **Condition:** drafts.length \> 0\.  
* **Layout:** Tinder-style Card Stack.  
* **Card UI:**  
  * **Top:** "Proposed Reply to \[Sender\]" (Serif).  
  * **Middle:** The summary of the incoming email (Sans).  
  * **Bottom:** The Draft text (Serif, inside a paper-textured box).  
  * **Footer:** "Reasoning Trace" (Collapsible).  
* **Gestures:**  
  * **Right:** Approve & Send.  
  * **Left:** Reject/Edit.

## **3.2. State B: Clear (Idle)**

* **Condition:** drafts.length \== 0\.  
* **Layout:** Single Static Card ("The Health Report").  
* **Content:**  
  * "All clear, Melissa."  
  * *Stats:* "14 emails batched. 0 pending."  
  * *Server:* "Node B is Sleeping (S3)."  
* **Actions:**  
  * **"Dismiss":** Closes modal.  
  * **"Deep Dive":** Navigates to app/admin/health.tsx for raw logs.

# **4\. NAVIGATION MAP**

1. **Home (The Dog House)** \- *Default State*  
   * Chat with Public/Private Model.  
   * View Avatar State.  
2. **Drawer (Settings)** \- *Slide from Left*  
   * Toggle Dark Mode.  
   * View House/Coop Feeds.  
3. **Deck (The Work)** \- *Slide from Bottom/Modal*  
   * Review Drafts.  
   * Review System Health (if clear).

# **5\. ASSET REQUIREMENTS (SUMMARY)**

* **avatar\_terrier.riv**: Rive animation (Idle, Alert, Typing).  
* **icon\_card\_stack\_bubble.svg**: The notification icon.  
* **watermark\_fetch.svg**: The background subtle logo.  
* **Font:** *Crimson Text* (or similar Serif) font file.
